Requirements:
  pip install flask pandas joblib flask-cors requests numpy
  npm install

Software Installations:

  Node.js – Required for running frontend tools or full-stack apps.
  
  Python Interpreter – Required to run your Python backend (Flask app).

  Chrome or Edge browser latest version

How To start :
Step 1: Open CMD
Step 2:  node server.js
         python app_api_integrated.py

Step 3: Open the Login.html
Step 4: Sign up 
Step 5: If hardware online you will see the sensor data
